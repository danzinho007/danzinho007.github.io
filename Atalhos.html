Atalhos PC

CTRL + F5 = Limpa os dados salvos do navegador
CTRL + < = Voltar a página anterior

Atalhos Visual Studio Code

Alt + Número
Navegar entre abas

Alt \/ ( seta pra baixo )
Mover linha pra baixo

Alt + Shift + F
Identação

Alt + Z
Trás a linha pra esquerda Pra colocar na direita é : selecionar o texto e apertar TAB

CTRL + Alt + Enter = Substituir arquivos

CTRL Click
Escolha uma palavra e segure CTRL e depois clique nela com o botão esquerdo do mouse, isso irá seguir
Caso não esteja criado o arquivo, vai perguntar se deseja fazer isso

CTRL Espaço
Seleciona algo na pasta

CTRL + Shift + P
Selecione algo e aperte essas teclas, escolha Envolver com abreviação e depois selecione sua tag
Ex : Esqueci de por negrito em Atalhos, dai eu faço isso e escolho a tag strong e aperto Enter
