A div "gameArea" é usada para controlar a área do jogo e a div "gameBox" é usada para exibir o jogo. O link https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.8.0/p5.js se refere a uma biblioteca JavaScript chamada P5.js.
P5.js é uma biblioteca de código aberto que fornece acesso aos recursos do Processing dentro do ambiente JavaScript.
É projetado para iniciantes e artistas, permitindo criar aplicativos interativos com facilidade, como jogos, animações e visualizações de dados.
A biblioteca é carregada na página HTML incluindo o link do arquivo p5.js em sua página HTML. A função draw é uma das funções nativas do p5.js, um framework JavaScript para desenho e animação.
Ela é chamada automaticamente pelo p5.js a uma taxa específica (60 vezes por segundo, por padrão) e contém todo o código de desenho e animação que você deseja exibir na tela. Em outras palavras, o que você escrever dentro da função draw será repetido a uma taxa definida, permitindo que você crie animações e jogue com formas, cores e imagens na tela. A função draw pode conter qualquer comando de desenho que você deseja usar.
Por exemplo, você pode desenhar formas, linhas, curvas, imagens, textos e muito mais, basta usar as funções de desenho apropriadas disponíveis no p5.js.
Além disso, a função draw pode acessar variáveis e funções definidas em outras partes do seu código, o que significa que você pode usá-la para atualizar as informações na tela de acordo com a lógica e as interações do jogo.

    reset(){
        clearInterval(this.timer); // resets the time
        this.a_dead.play();
        this.start(); // restarts the game
        
    }

    keyEvents(e){
        // goes left
        if(e.keyCode === 37 && this.speedX !== 1){
            this.a_start.play();
            this.speedX = -1;
            this.speedY = 0;
            this.frame = true;
        }
        // goes right
        if(e.keyCode === 39 && this.speedX !== -1){
            this.a_start.play();
            this.speedX = 1;
            this.speedY = 0;
            this.frame = true;
        }
        // goes down
        if(e.keyCode === 40 && this.speedY !== -1){
            this.a_start.play();
            this.speedX = 0;
            this.speedY = 1;
            this.frame = true;
        }
        // goes up
        if(e.keyCode === 38 && this.speedY !== 1){
            this.a_start.play();
            this.speedX = 0;
            this.speedY = -1;
            this.frame = true;
        }
    }
