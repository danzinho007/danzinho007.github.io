    A maioria das tags tem < abertura e fechamento >

    obs¹ = Isso são tags que precisam estar juntas
    Por exemplo : < !DOCTYPE HTML>, se eu juntar o <, não irá aparecer o código, por isso
     estou notificando antes
    obs² = Mensagem / Texto = A pessoa pode escrever qualquer coisa 

    Página em HMTL :

    1¹- < !DOCTYPE HTML>
    Diz pro navegador que o Documento está no formato HMTL
    
    2¹- < html lang="pt-br">
    pt-br ta dizendo pro navegador que é português do Brasil

    3¹- < head>
    E o cabeçalho do documento, é o visual do site
    
    4¹- < meta charset="UTF-8">
    Diz pro Site que ele permite o Conjunto de Caracteres UTF-8 ( Acentuação e Cecidilha ç )
    Meta não tem fechamento, temos 2 opções : Sem a barra no final é
    padrão html antigo ( xml ) e com a barra no final é padrão novo 

    5¹ < meta http-equiv="X-UA-Compatible" content="IE=edge">
    Indica ao Navegador IE que deve usar a versão mais recente

    6¹ < meta name="viewport" content="width=device-width, initial-scale=0.5">
    Vai ocupar 150% (0.5) da Tela do PC
    Se fosse 1.0 seria 100%
    Se fosse 1.5 seria 50%

    8¹ < title> Texto < /title>
    O que irá aparecer na Aba do Navegador

    < title>Meu site
    < link rel="stylesheet" href="estilo.css">
    Aqui eu to falando pro navegador que as folhas de estilo ( stylesheet ) está no arquivo estilo.css

    < link rel="shortcut icon" href="favicon.ico">
    Emoji na aba do navegador
    
    < style> < /style>
    style é uma tag de estilo da página
    Nesse caso estou usando estilo interno, porém também existe o estilo inline e externo
    O mais recomendado é o externo

    10¹ < body>
    É o corpo do site

    11¹ < h1> < /h1 >
    É o Título 

    12¹ < h2> < /h2 > até < h6> < / h6>
    São Subtítulos 
    
    13¹ < p> < /p>
    São parágrafos

    14¹ < script> < /script>
    São scripts
    Existe o script interno e externo, esse é o inteiro
    O mais recomendado é o externo
