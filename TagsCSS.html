Está definindo a cor de fundo do site pra rgb(169, 228, 178)

yellow, green, etc

rgb(35, 137, 195)
rgba (0,0,0,0) onde o último 0 é transparência pode ser 0 até 1
hsl(M, S%, L%) = MatizSaturaçãoLuminosidade*
hsla ( M, S%, L%, A) onde A é Alpha

04-margin: Xpx Ypx Wpx Zpx;
Margem : Parte externa de algo
Parte em volta dos conteúdos
Margem de cima, direita, baixo e esquerda
Se colocar -210 em Xpx ( margem de cima ) vai ficar lá encima a margem
Quanto maior o valor em Ypx ( margem da direita ) mais ela vai encolher pra esquerda
Colocando 400 em Zpx ( margem esquerda ) fica muito pra direita
Recomendo usar o site do mozila

06-src: url('nomedoarquivo.ttf');
O certo nem é 'nomedoarquivo.ttf', mas sim local onde está localizado a fonte + barra ( / ) nomedafonte.ttf
Nesse caso que coloquei acima a fonte está localizada na pasta do projeto, então ficou assim
Caso o arquivo esteja acima, basta colocar .. ( 2 pontos ) pra voltar pra pasta anterior
Exemplo: Estamos na pasta css no arquivo estilo.css, quero carregar algo que está na página fontes, eu coloco
: ../fontes/nomedoaquivo.extensão
No caso de site tipo o Github, não será possível ler pasta, por isso tudo precisa estar na mesma pasta
