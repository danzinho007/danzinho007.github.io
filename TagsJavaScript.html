onclick

Quando o usário clicar vai acontecer algo. Exemplo : onlick ="quebraLampada" Quando o usuário clicar vai chamar a função quebraLampada

onmousemove

Quando o usuário passar o mouse sobre o objeto, vai disparar algo Exemplo : onmousemove = "acendeLampada"
Quando o usário passar o mouse vai chamar a função acendeLampada

onmouseout

Quando o usuário tirar o mouse do objeto, vai acontecer algo
Exemplo : onmouseout = "lampada-apagada"
Quando o usuário tirar o mouse vai chamar a função lampada-apagada
